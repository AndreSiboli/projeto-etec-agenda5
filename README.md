# ⚡ Consumo de Energia

## 📌 Sobre o projeto
Esse projeto é um programa simples em Python que calcula o consumo mensal de energia de um aparelho, com base na potência e no tempo de uso diário.  A ideia é ter uma noção de quanto um aparelho gasta por mês.

## 🎯 Objetivo
Calcular o consumo em kWh/mês e mostrar também um custo estimado.

## 🛠️ Linguagem usada
- Python 🐍

## 🧮 Fórmula utilizada

```python
consumoMensal = (potencia * horasDia * 30) / 1000
```

## ▶️ Como executar
1. Abra o terminal na pasta do projeto  
2. Execute o comando:

```bash
python app.py
```

3. Preencha as informações pedidas:
- Nome do aparelho
- Potência (W)
- Tempo de uso diário (horas)

## 💡 Exemplo de saída

Aparelho: Geladeira </br>
Consumo estimado: 45.00 kWh/mês </br>
Custo estimado: R$ 33.75
